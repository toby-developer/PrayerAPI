# PrayerAPI
Simple API to get Prayer Times in Uzbekistan throughout various regions.
### Usage

```
/get/message/?region_id=43&month=2&day=2&language=uz

/get/prayer_times/?region_id=43&month=2&day=2&language=uz

/get/monthly_prayer_times/?region_id=43&month=2&language=uz
```
